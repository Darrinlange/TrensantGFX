Welcome to TGW - the Trensant Graphics Wrapper library

The Trensant Graphics Library is a wrapper of several 3rd party graphics and charting libraries which are useful for visualizing data using different techniques. See the README.md file in the distribution for more on which libraries and charts are wrapped and how to use them.


Why a chart wrapper?

This wrapper library supplies only a few charts natively but "wraps" numerous charts from many other popular libraries in to a common API and with simplified loading. This allows a programmer whose first langauge may not be javacsript and HTML to quickly use some of the charts here in a web page. For example for programmers used to Python, R or even C++ just drop your JSON data in a local folder, include a path to libs folder and then a minimal HTML page can be used to render some charts and graphics. In addition if you're just trying to get some charts running quickly this library can save much time since it handles a lot of "boiler plate code" and allows comparisons of different libaries charts using this wrappers common API.
Not every chart in each library is wrapped and in many cases "simplified" deciscions were made to get the chart to render with a minimal API. While this may not be optimal its easy to go the wrapper code and copy out native access to particular libary and chart and use that as base.
